WebTeX Fraction Fallback Test

This page tests the fallback system for fraction content that might cause KaTeX font errors like "No character metrics for '½'".

Note: The extension should automatically detect fraction patterns and use the custom renderer to avoid font-related KaTeX errors.

Basic Fractions (Font Error Prone)

These fractions might cause KaTeX font errors and should trigger the fallback system.
Simple fractions:
$\frac{1}{2}$ - One half
$\frac{1}{3}$ - One third
$\frac{2}{3}$ - Two thirds
$\frac{1}{4}$ - One quarter
$\frac{3}{4}$ - Three quarters
Expected: These should render as proper fractions without font errors.

Mixed Content with Fractions

Testing fractions mixed with other mathematical expressions.
Mixed expressions:
The probability is $\frac{1}{2}$ or 50%.
The area is $\frac{1}{4}$ of the total: $A = \frac{1}{4} \times \pi r^2$

For $\frac{1}{3}$ of the time, $x = \frac{2}{3}$
Expected: Fractions should render properly alongside other math expressions.

Fraction in Display Math

Testing fractions in display math mode.
Display fractions:
$$\frac{1}{2} + \frac{1}{3} = \frac{5}{6}$$
Complex display fraction:
$$\frac{\frac{1}{2} + \frac{1}{3}}{\frac{1}{4} + \frac{1}{5}} = \frac{\frac{5}{6}}{\frac{9}{20}} = \frac{50}{27}$$

Font Error Prevention

The extension should prevent these specific KaTeX errors:
  • "No character metrics for '½' in style 'Main-Regular' and mode 'text'"
  • "No character metrics for '⅓' in style 'Main-Regular' and mode 'text'"
  • Any other font-related character metric errors
