WebTex Extension Test Page

Testing Instructions:
  1. Open this page in Chrome
  2. Click the WebTex extension icon in your toolbar
  3. Toggle the switch to "ON" for this site
  4. Refresh the page to see LaTeX rendering
  5. Check the browser console for debug messages

Inline Math Examples

$E = mc^2$

Einstein's famous equation: $E = mc^2$ relates energy and mass.

\(a^2 + b^2 = c^2\)

The Pythagorean theorem: \(a^2 + b^2 = c^2\) is fundamental in geometry.

$\frac{d}{dx}x^n = nx^{n-1}$

The power rule for derivatives: $\frac{d}{dx}x^n = nx^{n-1}$

Display Math Examples

$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$

The Gaussian integral:

$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$
\[\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e\]

Definition of Euler's number:

\[\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e\]

Debug Information

Open the browser console (F12) to see debug messages from the WebTex extension.

You should see messages like:

  • "WebTex: Content script loaded for domain: localhost"
  • "WebTex: Domain check - localhost Enabled: true/false"
  • "WebTex: MathJax loaded successfully" (if enabled)
