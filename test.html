WebTex Extension Test Page

This page contains various LaTeX expressions to test the WebTex Chrome extension. Make sure the extension is enabled to see the rendered math.

Inline Math Examples

$E = mc^2$

Einstein's famous equation: $E = mc^2$ relates energy and mass.

\(a^2 + b^2 = c^2\)

The Pythagorean theorem: \(a^2 + b^2 = c^2\) is fundamental in geometry.

$\frac{d}{dx}x^n = nx^{n-1}$

The power rule for derivatives: $\frac{d}{dx}x^n = nx^{n-1}$

\(\sum_{i=1}^{n} i = \frac{n(n+1)}{2}\)

Sum of first n natural numbers: \(\sum_{i=1}^{n} i = \frac{n(n+1)}{2}\)
