Testing Instructions:

  2. Go to the Console tab
  3. Click the WebTex extension icon
  4. Try toggling the switch
  5. Watch the console for debug messages
  6. Check if the badge on the extension icon changes

Expected Console Messages

When you click the toggle, you should see messages like:

  • WebTex: Toggle clicked, new state: true
  • WebTex: Updated allowed domains: ["localhost"]
  • WebTex: Message sent to content script
  • WebTex: Storage changed: {allowedDomains: {newValue: ["localhost"]}}

Troubleshooting

If the toggle doesn't work:

  • Check if the extension is properly loaded in chrome://extensions/
  • Make sure "Developer mode" is enabled
  • Try reloading the extension
  • Check the console for error messages
  • Try the "Clear Storage" button above
